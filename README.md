# Алгоритм Флойда
  -это программа на Java, реализующая алгоритм Флойда-Уоршелла для поиска кратчайших путей между всеми парами вершин в взвешенном графе.

Для кого предназначена:
    Студенты, изучающие алгоритмы на графах
    Разработчики, нуждающиеся в базовой реализации алгоритма Флойда-Уоршелла
    Преподаватели для демонстрации работы алгоритма

Основные функции:
    Вычисление кратчайших расстояний между всеми парами вершин
    Восстановление полных путей между любыми вершинами
    Визуализация матриц расстояний и промежуточных вершин
    Детальный анализ всех возможных маршрутов в графе

Установка и запуск:
    Скачайте IntelliJ IDEA
    Откройте проект в IntelliJ IDEA, Eclipse или другой Java-совместимой IDE
    Импортируйте все файлы .java
    Запустите класс Main

Примеры использования:
    Для запуска программы:
    public static void main(String[] args) {
    double[][] graph = GraphUtils.createTestGraph();
    FloydAlgorith.FloydResult result = FloydAlgorith.mayweather(graph);
    // Вывод результатов
    GraphUtils.printMatrix(result.getDistances(), "Кратчайшие расстояния:");
  }

Получение конкретного пути:
    // Получение пути из вершины 1 в вершину 4
    List<Integer> path = FloydAlgorith.getPath(0, 3, result.getNextVertices());
    System.out.println("Путь: " + path.stream().map(v -> v + 1).toList());
    
 Детальный анализ:
   // Полный анализ всех путей

Струкутура репозитория:
    Main.java                  Главный класс, точка входа в программу
    FloydAlgorith.java         Реализация алгоритма Флойда-Уоршелла
    GraphUtils.java            Утилиты для работы с графами
    PathAnalyzer.java          Анализ путей и маршрутов
