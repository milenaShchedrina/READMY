# План тестирования AlgoritmFloyida

## Области тестирования:
1. Функциональность алгоритма
2. Интеграция компонентов
3. Производительность
4. Удобство использования

## Критерии успеха:
- 100% прохождение функциональных тестов
- Время выполнения для графа 100x1 < 5 секунд
- Положительные отзывы от 80% тестировщиков

## Ответственные:
- Разработчик: создание тестов
- QA: выполнение тестов
- Пользователи: обратная связь
